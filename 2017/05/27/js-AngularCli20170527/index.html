Angular4.0来了，更小，更快，改动少

接下来为Angular4.0准备环境和学会使用Angular cli项目

环境准备

1）在开始工作之前我们必须设置好开发环境

如果你的机器上还没有安装Node.js和npm，请安装他们

（这里特别推荐使用淘宝的镜像cnpm，记得以后把npm的指令改为cnpm就可以了）

然后我们可以通过node -v和cnpm -v来分别查看node和cnpm安装的版本和结果

2）安装全局Angular cli
